# Superstore Sales Analysis

## Project Purpose
This project analyzes retail sales data to identify profit patterns, best-selling categories, and loss-making regions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## Key Questions
- Which product categories generate the most profit?
- Which cities are operating at a loss?
- Monthly sales trend analysis

## Outcome
The analysis reveals regional performance differences and helps understand which segments should be prioritized for profitab
